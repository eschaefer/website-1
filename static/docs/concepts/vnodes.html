Virtual Nodes

Virtual Nodes

A virtual node is a JavaScript object that represents an element in the DOM tree.

{
  tag: "div",
  props: {
    id: "app"
  },
  children: [{
    tag: "h1",
    data: {},
    children: ["Hi."]
  }]
}

To create a virtual node, use the h function.

const node = h("div", { id: "app" }, [h("h1", {}, "Hi.")])

Alternatives to the built-in h() function include JSX, hyperx, t7 and @hyperapp/html.

const node = (
  <div id="app">
    <h1>Hi.</h1>
  </div>
)

Virtual node props may include any valid HTML or SVG attributes, DOM, lifecycle events, and keys.

const button = (
  <button
    class="ui button"
    tabindex={0}
    style={{
      fontSize: "3em"
    }}
    onclick={() => {
      // ...
    }}
  >
    Click Me
  </button>
)

A function that returns a virtual node is known as a component.
